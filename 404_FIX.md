### 2. Fix Dynamic Routes (Next.js 15+)

**Vấn đề**: Trong Next.js 15+, `params` là Promise

**Trước**:
```typescript
params: { slug: string }
// Sử dụng: params.slug
```

**Sau**:
```typescript
params: Promise<{ slug: string }>
const { slug } = await params
// Sử dụng: slug
```
