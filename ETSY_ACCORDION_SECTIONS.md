## 📋 SECTIONS CHI TIẾT

### 1. This Pattern Includes

**Content** (Bullet list):
- Detailed, beginner-friendly instructions with clear illustrations
- Nested Print at Home Files (US Letter + A4 compatible) with layers
- Large Format File (A0/Copyshop)
- Projector File
- Detailed Video Tutorial

**Style**: Bullet points với icon

### 2. Sizing

**Content**:
```
This pattern includes multiple size options to fit your needs. 
Detailed measurements are provided in the PDF file.
```

**Purpose**: Thông tin về kích cỡ

### 3. Suitable Fabrics

**Content**:
```
Recommended fabrics: Cotton, linen, canvas, denim, or any 
medium-weight woven fabric. Avoid stretchy fabrics unless 
specified in the pattern.
```

**Purpose**: Hướng dẫn chọn vải

### 4. Item Details

**Content**: 
```
{pattern.long_description || pattern.description}
```

**Purpose**: Mô tả chi tiết sản phẩm (dynamic)

### 5. Delivery

**Content**:
```
Instant Download

Your files will be available to download once payment is confirmed.

Instant download items don't accept returns, exchanges or 
cancellations. Please contact us about any problems with your order.
```

**Purpose**: Thông tin giao hàng & chính sách

### 6. Disclaimer

**Content**:
```
This is a digital sewing pattern, not a finished product. 
You will receive a PDF file to download and print. 
No physical items will be shipped.

All patterns are for personal use only. Commercial use 
requires a separate license.
```

**Purpose**: Tuyên bố miễn trừ trách nhiệm
